CRUD Carros

    Sobre o Sistema

    Esse é um projeto desenvolvido para a disciplina de Desenvolvimento de Sistemas Web 1 (DSW 1), evoluindo um CRUD básico de carros para uma arquitetura mais robusta, separando claramente front-end e back-end.

    O backend é uma aplicação Java empacotada como WAR, organizada em camadas seguindo o padrão MVC: modelos de entidade (Carro, Usuario), DAOs responsáveis pela persistência em arquivo, e Servlets que expõem endpoints REST retornando dados em JSON.

    O frontend, por sua vez, é composto apenas por páginas HTML e arquivos JavaScript puros, sem nenhuma dependência de JSP ou código Java. Toda comunicação com o servidor é feita de forma assíncrona através da Fetch API, sem a necessidade de recarregar a página a cada operação.

    O sistema permite cadastrar, listar, visualizar, editar, excluir e avaliar carros. Algumas ações — como adicionar, editar e remover registros — exigem que o usuário esteja autenticado, com o controle de sessão sendo gerenciado tanto no servidor (via HttpSession) quanto no navegador (via sessionStorage).
