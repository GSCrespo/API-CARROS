    Sobre o projeto

    Esse é um mini projeto desenvolvido para a disciplina de Desenvolvimento de Sistemas Web 1 (DSW 1). A aplicação foi construída utilizando JSP, com uma estrutura simples baseada em Servlets e listas em memória para armazenamento dos dados.

    O objetivo principal é demonstrar um CRUD básico de carros, permitindo cadastrar, listar, editar e excluir registros de forma prática, focando no entendimento do fluxo entre front-end e back-end.
